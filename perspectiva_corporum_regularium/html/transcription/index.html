PERSPECTIVA. Coporum Regularium. Das tift / ein flennige furmenfung/ Bie die funff Regulirten Coiper / Daruon Plato inn Timaeo / Bund Euclides inn sein Flementis Schzeibt / zc. Durch einen fonderlichen / nemen / behenden und gerechten weg / Der noz nie im gebzauch ift gesehen mozden / ger kunsclich in die Perspectiva gebracht / Und darzu cin schone Unlentung / wie auf denselbigen funff Cozpern one Endt / gar viel andere Cozper / mancherley urt und gestalt / gemacht / unnd gefunden werden mugen. Allen Liebhabern Der frenen Kunst zu Ehin / Durch Wenkeln Jamitzer / burgern und goldtschmit in Nurmberg / mit Goltlicher hulff an tag geben. z. c. Wit Kom: Kanserlicher Man: befrenung / Inn Jaren nicht nach zudzuden. Anno, M. D. LXVIII. Early state with no printed text. Full book: http://digital.slub-dresden.de/id271702370
